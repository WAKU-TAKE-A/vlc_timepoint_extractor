## [0.9.5] - 2026-01-14
### Added
- **ボタン操作時の選択チェック機能**: `Jump To`、`Update Remark`、`Remove Point` の各ボタン押下時、リストが未選択であればステータス欄に「Select a point first.」と警告を表示するように改善。

## [0.9.4] - 2026-01-14
### Changed
- **出力フォルダ名の統一**: 抽出データの保存先を `_frames` / `_cuts` から、より明示的な `_extracted_frames` / `_extracted_movies` へ変更。

## [0.9.3] - 2026-01-14
### Changed
- **GUIレイアウトの最適化**: `Extraction Settings` の開始行を 22 行目から 5 行目（操作ボタンの直後）へ移動し、縦方向の余白を削減。
- **リスト表示の高さ調整**: 設定項目の移動に伴い、右側リストの `row_span` を 16 に調整し、フッターボタンと高さを統一。

## [0.9.2] - 2026-01-14
### Changed
- **GUIレイアウトの定数化**: レイアウト計算の複雑さを排除するため、行・列の指定をマジックナンバー（直接数値）による絶対指定に変更し、目視での調整を容易化。
- **リスト表示のバグ修正**: リストが画面下部まで届かず途切れていた問題を、`row_span` を拡大することで修正。

## [0.9.1] - 2026-01-14
### Added
- **FFmpeg 存在確認**: 起動時および抽出時に、システム PATH に FFmpeg が存在するか自動チェックし、未検出時は UI 上に警告を表示する機能を追加。
- **UX改善 (Remark自動同期)**: TimePoint を選択して `Jump To` をクリックした際、その項目の備考（Remark）を入力欄に自動反映する機能を実装。
### Changed
- **ステート管理の強化**: 動画ファイルの切り替え（URI変更）を正確に検知し、別の動画データが残らないよう同期処理を刷新。
- **ファイル名サニタイズの改良**: 備考欄の内容をファイル名に使用する際、記号をアンダースコアに置換し、連続するアンダースコアをクリーンアップする処理を追加。
- **ボタン名の修正**: `EXTRACT FRAMES` を `Extract Frames` に変更し、UI の一貫性を向上。
### Security
- **データ読み込みの保護**: 外部 `.tp` ファイルの読み込みに `pcall` を導入。ファイルが破損している場合でも VLC 本体のクラッシュを防ぎ、安全に状態をリセットするよう改善。
### Fixed
- 入力なしの状態で一部のハンドラを実行した際のエラーハンドリングを改善。
